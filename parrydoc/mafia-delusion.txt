マフィア妄想
RDATAには、Parryのマフィアについての妄想に関する仕組みが書かれているようだ。

キーとなる語を、FLARELIST, SETLIST, SENSITIVEにわけて、
　　各LISTのproperty SETS にはconceptと呼ばれる言葉のlistが定義され
　　各conceptの各言葉のproperty WORDS にはそのconceptに属する単語が登録されている
　　また、conceptのproperty WTにはその重みが登録されている。

flarelistはマフィア妄想にいたるconceptの集合、setlistは一般的な関心事(たぶん)についてのconcept、
sensitivelisにはセンシティブな言葉のconceptの集合である。

マフィア妄想(Mafia delusion)というのがParryの重要な執着らしい。
BELにはDDMAFIAという状態があり、これがドクターがマフィアの一員だという確信を示していて
このファイルで定義している言葉の関係とBELの定義する状態との間をつないでいるのだろう。
たぶんPMEM4のevaluateとかproveとかがその処理にあたると思う。


また、各リストの中で、conceptには重みがつけられる。

FLARELIST  (SETS)   WORDS
　RACKETSET(17)      => RACKETER  CRIME
　GANGSTERSET(15)    => GANGSTERS HOOD
　PERSONSET(12)      => ALIOTO MAFIO
　CHEATSET (10)      => HEATING CHEAT
　BOOKIESET(9)       => BOOKIES CROOK
　GAMBLERSET(7)      => GAMBLING BET
　MONEYSET(6)        => MONEY
　HORSERACINGSET (5) => HORSERACING RACES
　POLICESET(4)       => POLICE FUZZ
　ITALIANSET(3)      => ITALIANS ITALY
　HORSESET(1)        => HORSES HORSE

たとえば、RACETSETのproperty WORDSに(RACKETER CRIME)が設定され、それは
RACKETERと CRIMEという語がconcept RACETSETに属する語だということになるらしい。
作り方からは、同じ語が別のconceptに属することはない。
()の中は重みで、property WTに設定される。
